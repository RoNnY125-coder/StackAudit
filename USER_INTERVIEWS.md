## Interview 1 — u/the-lost-soul, Independent Developer, Solo
Date: 2026-05-09
Duration: ~10 minutes
Platform: Reddit DM

**Background:**
Solo developer, previously had a small team. Currently pays for Cursor and Claude Pro. Uses both tools daily for development work but admits usage is inconsistent — some weeks he's in them constantly, other weeks barely at all.

**Direct Quotes:**
- "I used to need this for the whole team but now it's just me"
- "Sometimes I go weeks without really using it properly"
- "I just keep paying because switching feels like effort"

**Most Surprising Thing They Said:**
He knew he was probably overpaying — Claude Pro at $20/month and Cursor Pro at $20/month for a single person with inconsistent usage — but hadn't cancelled either because cancellation itself felt like more work than just letting the invoice run. The barrier wasn't cost awareness, it was switching friction.

**What It Changed About the Design:**
The audit results need a direct link next to each recommendation pointing to the plan downgrade page for that tool. Telling someone they're overpaying is not enough if acting on it still feels like effort. Also identified a specific user segment — solo developers who downsized from a team and are still paying team-era prices.
